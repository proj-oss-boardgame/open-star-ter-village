Public Money Public Code は、2017 年に欧州自由ソフトウェア財団（FSFE）が提唱したキャンペーンです。

税金で開発された政府システムはオープンソースとして公開すべきであると主張し、重複開発の防止と公共利益の最大化を目指しています。

[FSFE 發起的連署網頁](https://publiccode.eu/zh-tw/)
