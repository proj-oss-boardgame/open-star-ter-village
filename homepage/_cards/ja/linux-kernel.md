Linux Kernel は、1991 年に Linus Torvalds によって開発されたオープンソースの Unix 系 OS カーネルです。

世界中の 12,000 人以上の開発者が貢献しており、現在の多くの Linux ディストリビューションの中核を担っています。

- [Linux Kernel 公式サイト](https://www.kernel.org/)
